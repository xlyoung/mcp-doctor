## 🔒 Security Scoring

Each server in the registry receives a quality score (0-100) based on:

- **Security (35%)** — Prompt injection, path traversal, credential safety
- **Maintenance (25%)** — Commit frequency, issue response, release cadence

- **Testing (15%)** — Test coverage, CI status, integration tests
- **Community (10%)** — Stars, contributors, ecosystem integrations
